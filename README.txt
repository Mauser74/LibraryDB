## Проектная работа
# Тема Система автоматизации работы библиотеки, учёта книг, пользователей, выдачи книг пользователям

Проект выполнен с использованием фреймворка Django. База данных для демонстрации работы включена в репозиторий.

Установка зависимостей осуществляется командой: pip install -r requirements.txt

Запуск сервера производится командой: python manage.py runserver
